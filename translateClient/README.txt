新建upload和download文件夹,将需要翻译的excel文件放到upload文件夹下
当前文件夹下shift+右键打开powershell执行：pip install requests
然后在命令行继续执行: python ./translate.py
跟着选项选择文件开始翻译吧

Install Python 3.7 or above, and remember to select "command line" option.
Create new "upload" and "download" folders, and place the Excel files that need to be translated in the "upload" folder.
Open PowerShell by pressing "Shift" key and right-clicking on the current folder, then execute "pip install requests".
Next, continue in the command line by executing "python ./translate.py".
Follow the options to select a file and start translating.
